# Adopt-Pets


使用Vue框架打造寵物認養平台的SPA系統，預計包含CRUD完整功能。

### 主要技術

Vue Router
VueX
Bootstrap Vue
Axios 串接API
使用json-server製作API文件，並部署到Heroku


### 目前功能

* 想領養寵物者 - 可點選待領養的寵物們，觀看詳細資料。

* 想送養寵物者 - 可點“送養”按鈕，填寫資訊會及時出現在首頁。
